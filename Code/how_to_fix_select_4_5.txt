Πρέπει να φτιάξουμε τον πίνακα recipes_by_tag να μην έχει μέσα set από tags, αλλά με partition key κάθε tag ως text data type να βάζουμε κάθε συνταγή που έχει αυτό το tag.

Άρα θα έχουμε partitions ίσα με τον συνολικό αριθμό των tags που το καθένα θα έχει τις συνταγές που έχουν το tag, με διπλότυπα συνταγών από tag σε tag αλλά αυτό δεν μας ενδιαφέρει.
